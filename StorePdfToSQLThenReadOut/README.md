# Sample - 上傳PDF並存入SQL然後讀出並在Browser顯示

### 功能
* 上傳PDF檔案並存入SQL
* 顯示存入的文件列表
* 在文件列表中, 點選**查看PDF**可在瀏覽器直接顯示PDF檔

### 修改內容
* 建立EF Model
* 建立Controller可處理View上傳的檔案
* 建立View可以建瀏覽器做檔案上傳
* 存入SQL
* 將PDF由SQL中讀出, 並顯示在瀏覽器上
